# Cursor Rules Todo App

Next.js + TRPCを使用したモノレポ構成のTodoアプリケーション

## 技術スタック

- TypeScript
- Next.js (App Router)
- TRPC
- Prisma
- Tailwind CSS
- Shadcn/UI

## 開発環境のセットアップ

### 必要要件

- Node.js 18.0.0以上
- pnpm 8.9.0以上

### インストール

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev
```

### 開発用コマンド

```bash
# 開発サーバーの起動
pnpm dev

# ビルド
pnpm build

# リント
pnpm biome:lint

# フォーマット
pnpm biome:format

# 型チェック
pnpm type-check
```
